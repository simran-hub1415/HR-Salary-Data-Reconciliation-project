# HR Salary Data Reconciliation Dashboard (Excel Project)

## Project Overview
This project compares employee salary data between two HR systems to identify mismatches and discrepancies.

## Tools Used
- Microsoft Excel
- VLOOKUP
- IF Formula
- COUNTIF
- Pivot Table
- Charts for Dashboard

## Key Features
- Auto-generated Employee IDs
- Salary comparison between System A and System B
- Error detection for mismatched salaries
- Summary metrics dashboard with charts

## Outcome
Successfully identified salary discrepancies and created a reconciliation dashboard for HR analysis.
